# 02 — ManifestView: página de exibição do YAML

Status: done

## O que fazer

- Criar `ManifestView` com rota `/yaml/{resourceType}/{namespace}/{name}`
- Exibir YAML em `<pre>` com fonte monospace
- Botão "Back" para retornar à listagem anterior
- Título com tipo e nome do recurso
- Tratar erros de conexão com notificação

## Critério de aceite

- Navegação via URL funciona
- YAML exibido corretamente
- Compilação sem erros
